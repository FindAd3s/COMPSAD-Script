Setup Instructions


1. Ensure you have python and pip, and XAMPP MySQL installed on your device

2. To install the necessary libraries needed for the code, run "pip install -r requirements.txt" (Python 2), or "pip3 install -r requirements.txt" (Python 3) [Any of these is fine]

3. Start Apache and MySQL on XAMPP

4. Create the necessary database titled 'fies" on MySQL

5. Running the script
    - Before running, save the excel file to ensure that the data is successfully uploaded.
    - Run 'uploadToMySQL.py' to publish the edits done onto the Excel file attached.
